# Nokia WMS — Redesign Project

## Folder Structure

```
project-root/
│
├── frontend/
│   ├── templates/     ← All HTML pages
│   ├── styling/       ← All CSS files
│   ├── scripts/       ← All JavaScript files
│   └── assets/
│       ├── images/
│       ├── icons/
│       ├── logos/     ← logo.png
│       └── fonts/
│
├── backend/           ← Reserved for future backend implementation
│
└── README.md
```

## Entry Point

Open `frontend/templates/dashboard.html` in a browser to start.

## Tech Stack

- HTML5 / CSS3 / Vanilla JS + jQuery
- Bootstrap 5.3
- Bootstrap Icons 1.11
- DataTables 1.13
- SweetAlert2

## Notes

- All navigation is client-side via relative paths.
- Backend directory is intentionally empty.
